In order to run the app, follow the following steps.

1. 1st Create a virtual environment and then activate it.
  
2. Run the requirements.txt file to install the required libraires
   pip install -r requirements.txt

3. Now, once all the libraries are install, run the demo.py file using command.

    python demo.py


4. You can open the messages.db in sqlite3 database application in order to verify the messages that are being stored.
